Triangular lattice barrier¶

authors: Joseph Marcus

Here I simulate genetic data under the coalescent in a triangular lattice with a uniform / symmetric migration and explore the fit of different ways to compute expected genetic distances on simulated genotypes.

Setup the habitat¶

Here we define a triangular lattice with 8 rows and 8 columns so we have 64 demes in total

Next we need to define a migration surface which is a function on the nodes of the graph that define edge weights. Here I choose a uniform and symmetric migration surface

In [13]:
def uniform_migration(self):
    """            
    Returns:
        g: nx.graph
            regular lattice with assigned weights
    """
    for i,j in self.g.edges():
        print(self.g.nodes)
        self.g[i][j]['m'] = 1.
    self.m = nx.adjacency_matrix(self.g, weight='m')
    self.m = self.m.toarray()
    #self.degree = np.count_nonzero(self.m, axis=1)[:, None]
    # normalize to sum to 1
    self.m = self.m / np.sum(self.m, axis=1, keepdims=True)
