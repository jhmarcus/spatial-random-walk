\section{Background}\label{background}}

    Laplacian smoothing is a way to smooth out observed values on the nodes
of the graph see

\begin{itemize}
\tightlist
\item
  http://www.stat.cmu.edu/\textasciitilde{}cshalizi/networks/16-1/lectures/13/lecture-13.pdf
\item
  http://www.stat.cmu.edu/\textasciitilde{}ryantibs/papers/sparsify.pdf
\end{itemize}

It can be conceived as the following optimization problem \ldots{}

\[
\underset{\mathbf{\tilde{\mathbf{y}}} \in \mathbb{R}^n}{min} \ ||\mathbf{y}-\tilde{\mathbf{y}}||^2_2 + \lambda \sum_{i,j} w_{ij}(\tilde{y}_i - \tilde{y}_j)^2 
\]

Where \(w_{ij}\) is the edge weight. We can represent this in matrix
form as \ldots{}

\[
\underset{\mathbf{\tilde{\mathbf{y}}} \in \mathbb{R}^n}{min} \ ||\mathbf{y}-\tilde{\mathbf{y}}||^2_2 + \lambda \tilde{\mathbf{y}}^T\mathbf{L} \tilde{\mathbf{y}}
\]

where \(\mathbf{L}\) is the graph laplacian. It can be shown the
solution to this problem is analytical \ldots{}

\[
\tilde{\mathbf{y}}^* = (\mathbf{I} + \lambda \mathbf{L})^{-1} \mathbf{y}
\]

\[
\mathbf{y} = (\mathbf{I} + \lambda \mathbf{L})\tilde{\mathbf{y}}^*
\]

Lets now use this to motivate the setup an autoregressive process
\ldots{}

\[
\mathbf{y} = (\mathbf{I} + \lambda\mathbf{L})\mathbf{y} + \nu
\]

where \(\nu \sim \mathcal{N}(\nu|\mathbf{0}, \sigma^2\mathbf{I})\)
\ldots{}

\[
\mathbf{y} - (\mathbf{I} + \lambda\mathbf{L})\mathbf{y}  = \nu
\]

\[
\mathbf{y}\big(\mathbf{I} - (\mathbf{I} + \lambda\mathbf{L})\big) = \nu
\]

\[
\mathbf{y} = \frac{1}{\lambda}\mathbf{L}^{-1}\nu
\]

This implies that \ldots{}

\[
\mathbf{y} | \lambda, \sigma^2, \mathbf{L} \sim \mathcal{N}\Big(\mathbf{y}|\mathbf{0}, \frac{\sigma^2}{\lambda}(\mathbf{L}\mathbf{L}^T)^{-1}\Big)
\]

Because \(\lambda\) and \(\sigma^2\) are unidentifiable let
\(\tau = \frac{\sigma^2}{\lambda}\) \ldots{}

\[
\mathbf{y} | \tau, \mathbf{L} \sim \mathcal{N}\big(\mathbf{y}|\mathbf{0}, \tau(\mathbf{L}\mathbf{L}^T)^{-1}\big)
\]

This is exactly the covariance derived in Hanks 2016 from the
perspective of a spatio-temporal random-walk. Perhaps this explains why
it performs the best in the
\href{https://github.com/jhmarcus/spatial-random-walk/blob/master/notebooks/isolation_by_srw.ipynb}{isolation\_by\_srw}
notebook? For more intution lets consider a two deme example (assuming
\(\lambda = 1\)) \ldots{}
